# Screens

## MVP screens
- **Dashboard/Home** — high-level overview of active batches and recent events.
- **Batch Detail** — selected batch state, timestamps, and action controls.
- **WIP/Drying Board** — in-progress and drying queues with status transitions.
- **Stops Log** — list of stops/incidents and their resolution status.
- **Role/Permission View** — basic access visibility for operators/admins.

## Later (post-MVP)
- **Analytics/Trends** — throughput, cycle time, stop-rate analysis.
- **Audit Explorer** — advanced filter/search across event history.
- **Admin Configuration** — feature flags and integration tuning UI.
